# Portfolio Website

A modern, responsive portfolio website for a 3rd year Computer Science student with freelance experience.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with blue and dark color scheme
- **Smooth Animations**: Engaging animations and transitions throughout
- **Interactive Elements**: Hover effects, scroll animations, and smooth navigation
- **Sections**:
  - Hero section with introduction
  - About section with statistics
  - Skills section with progress bars
  - Projects showcase
  - Experience timeline
  - Contact form
  - Social media links

## Color Scheme

- Primary Blue: #0066ff
- Light Blue: #3d8bff
- Dark Blue: #0047b3
- Background Dark: #0a0e1a
- Accent: #00d9ff

## Customization

### Update Personal Information

1. Open `index.html`
2. Update the following sections:
   - Name in the hero section
   - About section text
   - Skills and their proficiency levels
   - Projects (add your own projects)
   - Experience timeline
   - Contact information

### Add Your Links

Replace the `#` placeholders with your actual social media and project links:

- GitHub profile
- LinkedIn profile
- Twitter/X profile
- Email address
- Project demo links
- Project repository links

### Change Colors

Edit `styles.css` and modify the CSS variables in the `:root` section:

```css
:root {
  --primary-blue: #0066ff;
  --light-blue: #3d8bff;
  /* ... other colors */
}
```

## Setup

1. Download all files (index.html, styles.css, script.js)
2. Place them in the same folder
3. Open `index.html` in your web browser
4. Customize with your information
